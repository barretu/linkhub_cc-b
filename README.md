# 📚 Linkhub CC-B · CESAR School

> Hub de informações da turma de **Ciência da Computação — Turma B 2026.1** da CESAR School. Reúne grade semanal, calendário de provas, monitorias, links do Classroom e links institucionais em um único lugar.

## Funcionalidades

- Grade semanal com destaque no dia atual
- Calendário de provas e monitorias agrupados por disciplina
- Contador regressivo para a próxima prova
- Links do Google Classroom por disciplina
- Modo escuro/claro com preferência salva
- QR Code do site com download em `.png`
- Instalável como app (PWA) com suporte offline

## Tecnologias

- HTML, CSS e JavaScript vanilla — sem frameworks, sem build
- CSS Grid com layout totalmente responsivo
- PWA com Service Worker (funciona offline)
- QuickChart API para geração de QR Code

## 📱 Instalando como app no celular

**Android (Chrome)**
1. Abra o site no Chrome
2. Toque nos três pontinhos ⋮ no canto superior direito
3. Selecione **"Adicionar à tela inicial"** ou **"Instalar app"**
4. Confirme tocando em **"Adicionar"**

**iPhone (Safari)**
1. Abra o site no Safari (não funciona em outros navegadores)
2. Toque no ícone de compartilhar — quadrado com seta pra cima, na barra inferior
3. Role e toque em **"Adicionar à Tela de Início"**
4. Confirme tocando em **"Adicionar"**
